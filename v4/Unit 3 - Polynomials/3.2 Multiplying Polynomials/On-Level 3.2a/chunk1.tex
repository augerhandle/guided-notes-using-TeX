\section{Multiplying Monomials $\times$ Monomials}
\subsection{Product Property of Exponents}

\myCenteredBox[width=6in,]{
    When multiplying two \gap{identical} bases with \gap{exponents},
    simplify this as
    the base raised to \gap{the sum} of the exponents:
    %
    {\LARGE
    \[
        b^n \, b^m = b ^ {(n+m)}
    \]
    }
}

What $b^n$ {\bfseries\itshape really means}:
\vspace{1in}

We can use this to simplify $b^3\,b^5$.
\vspace{3in}

\myProblems[Multiply these monomials.]
    {
        $ (2x^{3}) \, (3x^{5}) $
    }
    {
        $ (3z^{2}) \, (4z^{3}) $
    }
    {1in}
\myProblems
    {
        $ (2x^2y) \, (3xy^2)  $
    }
    {
        $ (5x^2y) \, (3x^4y^2) $
    }
