\section{Transforming Reciprocal Functions}

The formula for a transformed reciprocal function is 
\vspace{-0.5\onelineskip}
\myCenteredBox[width=3.5in,valign=center]{
    \[ f(x) = \frac{\bm{a}}{x-\bm{h}} + \bm{k} \]
    \centering{\tiny($\bm{h}$ is the \gap{opposite of what you see})}
}

\vspace{-0.5\onelineskip}

The constants $\bm{a}$, $\bm{h}$, $\bm{k}$ have the usual effects:
\begin{itemize}[nosep]
    \item \gap{tranformations} of the graph
    \item changes to the \gap{attributes}
\end{itemize}
