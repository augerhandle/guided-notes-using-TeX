\section{Introduction}

A rational function is a \gap{fraction} of \gap{polynomials} like this 
    \begin{align*}
        f(x) &=  \frac{x^3 + 2x^2 - 5x - 6}{x^2 + 3x - 10}
    \end{align*}

\begin{tcolorbox}[center,width=4.5in,colback=white,]
    We will be finding these attributes of rational functions:
    \begin{itemize}[nosep]
        \item the $y$-\gap{intercept},
        \item the 
            \gap{horizontal} 
            \whenHONORS{or \gap{slant}} 
            \myEmph{asymptote},
        \item the \gap{domain},
        \item the \gap{holes},
        \item the \gap{vertical} \myEmph{asymptotes}, and
        \item the $x$-\gap{intercepts}.
    \end{itemize}
\end{tcolorbox}
