\section{Completing the Square}

Sometimes we want to rewrite trinomials that look like this: 
\gap{$ax^2 + bx + c$} 
so that they look like this instead: \gap{$(x-h)^2 + k$},
which is the sum of
\begin{itemize}[nosep]
    \item the \gap{square} \gap{of} \gap{a} \gap{binomial} and 
    \item a \gap{number}.
\end{itemize}

To do that we do what is called \gap{completing} \gap{the} \gap{square}.

\begin{myConceptSteps}{To rewrite a quadratic trinomial by completing the square\dots}
    \myStep{middle term}{%
        Take \gap{$\frac{1}{2}$} the \gap{coefficient} of the middle term, 
        and then  \gap{square} the result.
        }
    \myStep{add/subtract}{%
        \gap{Add} and \gap{subtract} the result of step 1 
        \gap{after} the middle term.
    }
    \myStep{group}{%
        Group the first \gap{three} terms. 
        \begin{itemize}[fullwidth,nosep]
            \item These will always be a \gap{perfect} \gap{square} \gap{trinomial}.
        \end{itemize}
        }
    \myStep{rewrite}{Rewrite the expression.
        \begin{itemize}
            \item Rewrite the group as the square of a binomial: \gap{$(x-h)^2$}.
            \item Combine the last three terms into a single number, \gap{$k$}.
        \end{itemize}
    }
\end{myConceptSteps}


\myProblems[Rewrite these trinomials by {\bfseries\itshape completing the square}.]
{
    $x^2 + 4x + 7$
}
{
    $x^2 - 8x -9 $
}
