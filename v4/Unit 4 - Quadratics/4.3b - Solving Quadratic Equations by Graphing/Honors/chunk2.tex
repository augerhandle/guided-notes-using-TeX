\begin{myConceptSteps}{To solve a quadratic equation by graphing\dots}
    \myStep{standard form}{%
        Rewrite the equation as \gap{$ax^2 + bx + c = 0$}.
        }
    \myStep{graph}{%
        Graph the function $f(x) = ax^2 + bx + c$.
        (We will use \myKey{Y=} on the {\scshape TI-84s}.)
    }
    \myStep{zeros}{%
        The zeros are the \gap{$x$-coordinates} of 
        the {$x$-intercepts} of the graph.
        }
    \myStep{solution}{%
        The {\bfseries\itshape zeros} are the solutions.
        {
            \Large
            \fbox{$\bm{x} =$\dots {\itshape zeros}\dots.}
        }
    }
\end{myConceptSteps}

We will find zeros using the \gap{TI-84} \gap{calculators}.
