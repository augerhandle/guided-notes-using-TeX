\section{Solving Linear Equations That Have Fractions}

If you have a fraction in an equation, 
get rid of it by \gap{multiplying} both sides of the equation 
by the \gap{denominator}.
(You might have to do this \gap{several} times if you have several fractions.)
\vspace{1\baselineskip}

\myProblems[Solve these equations.]
{
    \centering 
    $3x + 5 = 25 + 2x$
}
{
    \centering
    $ 5(k+2) = 11 + 4k$
}
{2.75in}

\myProblems
{
    \centering
    $ 2 - 8x - 7 = -2x + 7 - 8x $
}
{
    \centering
    $ -6m - 5(1+5m) = 31 + 5m$
}
{3in}
