Some problems might ask you to \emph{evaluate} an expression
with variables in it 
by telling you the values of all the variables.
Again, the word ``evaluate'' is a hint to you that your answer should be a \emph{number}.

\begin{myConceptSteps}{To evaluate expressions given the value of all the variables\dots}
    \myStep{substitute}{
        Substitute the value of each variable into the expression wherever that variable occurs
        It is a good idea to use parentheses wrapped around the value.
        This will give you an expression with no variables.
    }
    \myStep{evaluate}{
        Evaluate the expression using GEMDAS.
    }
\end{myConceptSteps}


\myBlankExample{2in}{
    Evaluate this expression:
    \(
        16 - 3(2x-1)^2 + 9
    \),
    where $x=2$.
}

\myBlankExample{2in}{
    Evaluate this expression:
    \(
        1 - \sqrt{b^2 - 4ac}
    \),
    where $a=-1$, $b=4$, $c=-3$.
}
