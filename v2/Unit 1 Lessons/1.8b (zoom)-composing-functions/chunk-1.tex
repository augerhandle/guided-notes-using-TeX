Yesterday we learned how to ``join'' two functions together ---
taking the output of the first one and passing that as an input to the second one.
This was called \emph{composing functions}.

But yesterday, our input was a number.
\begin{taggedblock}{on-level}
    Today it will be a variable.
\end{taggedblock}
\begin{taggedblock}{pre-AP}
    Today it will be a variable or an algebraic expression.
\end{taggedblock}

\begin{myConceptSteps}{To evaluate a composite function\dots}
    \myStep{input}{Pass the input to the {\bfseries\itshape nearest function} on the left.}
    \myStep{evaluate}{Evaluate that function. (You will get an algebraic expression.)}
    \myStep{pass}{Pass that expression to the next function to the left.}
    \myStep{evaluate}{Evaluate the that next function. (You will get another expression.) This is the output of the composite function.}
\end{myConceptSteps}

\begin{center}
    \begin{tcolorbox}[width=3.5in]
        \(   (f\circ g)(x) = f( g(x))    \)
        \vskip1em
        Pass the input to $g$ first (because $g$ is closest to $x$).
        Then pass the result of $g(x)$ to $f$.
    \end{tcolorbox}
\end{center}

\begin{center}
    \begin{tcolorbox}[width=3.5in]
        \(   (g\circ f)(x) = g( f(x))    \)
        \vskip1em
        Pass the input to $f$ first (because $f$ is closest to $x$).
        Then pass the result of $f(x)$ to $g$.
    \end{tcolorbox}
\end{center}
