

\section*{Imaginary numbers}

You can easily calculate the following square roots  
(in your head or using a calculator) :
\begin{itemize}[itemsep=0in]
    \item $\sqrt{25}$
    \item $\sqrt{4}$
    \item $\sqrt{121}$
    \item $\sqrt{1}$
    \item $\sqrt{123}$
\end{itemize}
But what happens when you take the square root of a negative number?

\myBlankExample{1.25in}{Calculate $\sqrt{-36}$}

Imaginary numbers are a {\bfseries\itshape new kind of number} that allow you to 
find the {square root} of a {negative} number.

\begin{center}
    \begin{tcolorbox}[width=5in]
        {\bfseries\itshape Imaginary numbers} are identified by the letter {\LARGE $i$}
        \vskip1em
        {
            \huge
            \begin{tcolorbox}[width=2in,height=4em,valign=center,colback=white,before=,after=\hfill]
                $i =$ \gap{$\sqrt{-1}$}
            \end{tcolorbox}
            \begin{tcolorbox}[width=2in,height=4em,valign=center,colback=white,before=,after=\hfill]
                $i^2 = $ \gap{$-1$}
            \end{tcolorbox}
        }
    \end{tcolorbox}
\end{center}

Using $i$,
we can find the square root of negative numbers, too.

\begin{taggedblock}{on-level}
    \myBlankExample{1in}{
        Find these square roots:
        \begin{itemize}[itemsep=0in]\Large
            \item $\sqrt{4}$
            \item $\sqrt{45}$
        \end{itemize}
    }
    
    \myBlankExample{1in}{
        Find these square roots:
        \begin{itemize}[itemsep=0in]\Large
            \item $\sqrt{-4}$
            \item $\sqrt{-45}$
        \end{itemize}
    }
\end{taggedblock}
