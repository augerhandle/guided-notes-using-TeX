\section*{Solving quadratic equations using the quadratic formula}

In the last lesson,
we determined the \gap{number} and \gap{type}
of solutions to quadratic equations using the
\gap{discriminant}---just a tiny part of the quadratic formula.
\vskip2em
\noindent
Today,
we will \gap{solve} quadratic equations
using the quadratic formula.

\begin{myConceptSteps}{
    To solve a quadratic equation using the quadratic formula\dots
}
\myStep{rewrite}{
    Collect all terms on \gap{one side} of the equation
    to get $\displaystyle ax^2 + bx + c = 0 $
}
\myStep{$a$, $b$, $c$}{
    Find $a$, $b$, and $c$ from the rewritten equation.
}
\myStep{discriminant}{
    Find $b^2-4ac$.
    (Do yourself a favor. 
    Calculate this {\bfseries\itshape before} working with the quadratic formula!)
}
\myStep{substitute}{Substitute $a$, $b$, $c$, and the {\itshape discriminant} into the quadratic formula.}
\myStep{evaluate}{Evaluate the right-hand-side.}
\myStep{solve}{
    You will have two solutions:
    \begin{itemize}[itemsep=1em]
        \item first solution:\qquad
            $\displaystyle x = \frac{-b + \sqrt{b^2-4ac}}{2a}$
        \item second solution:\quad
            $\displaystyle x = \frac{-b - \sqrt{b^2-4ac}}{2a}$
    \end{itemize}
    When $b^2 - 4ac =$ \gap{$0$}, these two solutions are \gap{the same}.
}
\end{myConceptSteps}
