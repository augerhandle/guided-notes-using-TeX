\section*{The Quadratic Formula}

We can \gap{solve} quadratic equations with something called
the ``quadratic formula''.

\begin{center}
\begin{tcolorbox}[width=5.5in]
    To solve the quadratic equation
    {\LARGE
        \[
            a x^2 + bx + c = 0
        \]
    }
    calculate $x$ using this formula:
    {\LARGE
        \[
            x = \frac
                {-b \pm \sqrt{b^2 - 4ac}}
                {2a}
        \]
    }
\end{tcolorbox}
\end{center}
