\section*{Converting standard form to vertex form by completing the square}

We can use completing the square from above
to convert any quadratic function in standard form ($ax^2 + bx +c$) into vertex form.

\begin{myConceptSteps}{
    To convert a quadratic function in standard form $(x^2+bx+c)$ to vertex form\dots
    }
    \myStep{factor out $a$}{
        If $a \neq 1$, factor it out of the trinomial.
        With $a$ factored out of the trinomial,
        we get a simpler trinomial where the coefficient on $x^2$ is 1.
    }
    \myStep{complete the square}{
        Complete the square on that simpler trinomial
    }
    \myStep{distribute $a$}{
        Distribute the $a$ back.
    }
\end{myConceptSteps}
    
\myBlankExample{3.5in}{
    Convert this quadratic function to vertex form by completing the square:
    \[y = 2x^2 + 4x -4 \]
}
    
\myBlankExample{3.5in}{
    Convert this quadratic function to vertex form by completing the square:
    \[y = -x^2 + 6x -2\]
}
