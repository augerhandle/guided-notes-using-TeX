

\section*{Introduction to quadratic functions}

This year, 
we have already studied two kinds of functions:
\begin{itemize}[itemsep=0in]
    \item \gap{linear} functions like $f(x) = 2x + 3y - 5$, and
    \item \gap{absolute value} functions like $f(x) = -2|x-3| + 4$.
\end{itemize}.
In this unit, we will study 
\gap{quadratic} functions.
Here are examples:
\begin{itemize}
    \item $f(x) = x^2$
    \item $f(x) = 7x^2$
    \item $f(x) = x^2 + 8x$
    \item $g(x) = 5x^2 + 3x - 4$
    \item $h(x) = -2(x-3)^2 + 4$.
\end{itemize}
