\section{Pieces of Parabolas}

Tomorrow we will start talking about their \gap{equations}.
Today  we will talk about the ``pieces'' of the parabolas:
\begin{itemize}[nosep]
    \item the \gap{focus}, $F$, which is a \gap{point},
    \item the \gap{directrix} $\mathscr{D}$, which is a \gap{line},
    \item the \gap{axis} \gap{of} \gap{symmetry} (AOS), which is a \gap{line},
    \item a parameter \gap{$p$}, which is a \gap{number},
    \item and the parabola itself.
\end{itemize}

            \item Draw the AOS
