\section*{Introduction}

We can \gap{solve} quadratic equations with something called
the \gap{quadratic} \gap{formula}.

\begin{tcolorbox}[center, width=6in]
    To solve a quadratic equation, $a x^2 + bx + c = 0$,
    calculate $x$ using this formula:
    {
        \large
        \[
            x = \frac
                {-b \pm \sqrt{b^2 - 4ac}}
                {2a}
        \]
    }
\end{tcolorbox}
