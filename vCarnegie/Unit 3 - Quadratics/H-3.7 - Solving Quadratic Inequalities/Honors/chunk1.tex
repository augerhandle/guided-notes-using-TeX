\section{Intervals}

The solution of a quadratic equation is \myEmph{two}, \myEmph{one}, or \myEmph{no} \gap{$x$ values} on the number line
(where the graph of the function \gap{crosses} the $x$-axis).

The solution of a quadratic inequality is a \gap{region} on the number line
(where the graph of the function is \gap{above} or \gap{below} and maybe \gap{on} the $x$-axis).
These regions are called \gap{intervals}.

There are three different \gap{notations} for intervals:
\begin{itemize}[nosep]
    \item intervals drawn on the number line,
    \item interval notation, and 
    \item inequalities.
\end{itemize}

        \begin{tcolorbox}
            \centering
            {
                \footnotesize
                \myAndInterval{white}\\[0.25\onelineskip]
            }
            $\left(L,R\right)$\\[0.75\onelineskip]
            $L < x < R$
        \end{tcolorbox}
        \begin{tcolorbox}
            \centering
            {
                \footnotesize
                \myAndInterval{black}\\[0.25\onelineskip]
            }
            $\left[L,R\,\right]$\\[0.75\onelineskip]
            $L \le x \le R$
        \end{tcolorbox}
        \begin{tcolorbox}
            \centering
            \myOrInterval{white}\\[0.25\onelineskip]
            $\left(-\infty,L\right) \enspace\bigcup\enspace \left(R,\infty\right)$\\[0.75\onelineskip]
            $x < L  \quad\text{OR}\quad x > R$
        \end{tcolorbox}
        \begin{tcolorbox}
            \centering
            \myOrInterval{black}\\[0.25\onelineskip]
            $\left(-\infty,L\,\right] \enspace\bigcup\enspace \left[\,R,\infty\right)$\\[0.75\onelineskip]
            $x \le L  \quad\text{OR}\quad x \ge R$
        \end{tcolorbox}
    \end{tcbraster}
}
