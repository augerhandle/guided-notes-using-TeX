\section{Multiplying Monomials}

To multiply polynomials, 
you need to multiply many \gap{monomials}.
Do this by applying the \gap{product} \gap{property} exponent rule.
\begin{tcolorbox}[center,width=2.5in]
    \Large\vspace{-1.2\onelineskip}
    \[
        B^n B^m = B^{n+m}
    \]
\end{tcolorbox}


\myProblems[Multiply the following monomials.]
    {
        $ (2x^3) (-3x^4) $
    }
    {
        $ \left(-\frac{1}{2}z^3\right) \left(-\frac{3}{2}z^2\right) $
    }
    {1.5in}
