Let's review how to \gap{simplify} rational expressions.

\begin{enumerate}[nosep]
    \item \myEmph{factor}
    \item \myEmph{domain} (excluded values)
    \item \myEmph{simplified expression} (cancel)
    \item \myEmph{result} (simplified expression + domain)
\end{enumerate}

{

    $

        {5x^2 - 10x} % 5x(x-2)
        {x^2 + 6x - 16} % (x-2)(x+8)
    $
}
