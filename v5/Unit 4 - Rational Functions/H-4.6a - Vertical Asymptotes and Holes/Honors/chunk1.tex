\section{Introduction}

To find \gap{vertical} \gap{asymptotes} and \gap{holes}, 
you need to be able to 
\begin{itemize}[nosep]
    \item \gap{factor} the numerator and denominator, and 
    \item find the \gap{excluded} values by checking \gap{denominator $\ne 0$.}
\end{itemize}

\myProblems[Factor these rational expressions, and find their excluded values]
{
    $
    \frac
        {2x^2 - 9x - 5}
        {3x^2 - 15x}
    $
}
{
    $
    \frac
        {3x^2 + 7x - 6}
        {6x^2 + 18}
    $
}
{2in}
[\myAnswer{$\frac{(2x+1)(x-5)}{3x(x-5)}$, $x \ne 0, 5$}]
[\myAnswer{$\frac{(x+3)(3x-2)}{6x(x+3)}$, $x \ne 0, -3$}]
