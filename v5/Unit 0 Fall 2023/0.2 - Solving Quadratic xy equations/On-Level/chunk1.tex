We can solve quadratic equations using 
the \gap{quadratic} \gap{formula}.

\hfill
\begin{minipage}{0.3\textwidth}
    \raggedleft
    \large
    $ax^2 + bx + c = 0$
\end{minipage}
\hfill
{
    \large $\Rightarrow$
}
\hfill
\begin{minipage}{0.3\textwidth}
    \raggedright
    \large
    \(
        x = \frac
            {-b \pm \sqrt{\text{\itshape discriminant}}}
            {2a}
    \)
\end{minipage}
\hfill\,
