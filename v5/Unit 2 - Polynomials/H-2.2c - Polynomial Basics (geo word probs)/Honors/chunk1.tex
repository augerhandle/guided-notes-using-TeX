\section{Given/To-Do/Solution Framework}

\gap{Word} \gap{problems} can be difficult,
because it's difficult to know how to start.
In this lesson you will learn a \gap{framework} 
that breaks word problems into smaller pieces 
and gives you a way to get started.

\vfil 

\begin{myConceptSteps}{To solve problems using the \myEmph{Given/To-Do/Solution} framework\dots}
    \myStep{Given}{
        \gap{Repeat} the information given in the word problem.
        \gap{Draw} a diagram.
        }
    \myStep{To-Do}{
        \gap{Restate} what you are supposed to find.
        }
    \myStep{Solution}{
        \gap{Solve} the problem by applying the lesson to the given information.
        }
\end{myConceptSteps}

\vfil 

\begin{tcolorbox}[center,width=5.5in]
    When I ask you to solve a problem using the 
    \myEmph{Given/To-Do/Solution} framework,
    write the title of each step like this:
    {
        \begin{adjustwidth}{2em}{0in}
            \fontfamily{augie}\selectfont 
            {\underline{Given:}   \hspace{0.2in}\huge\dots}\\[1\onelineskip]
            {\underline{To-Do:}   \hspace{0.2in}\huge\dots}\\[1\onelineskip]
            {\underline{Solution:}\hspace{0.2in}\huge\dots}
        \end{adjustwidth}
    }
\end{tcolorbox}
