\section{Vertex Form}

For us, quadratic functions in vertex form will always look 
like this.
\begin{tcolorbox}[center,width=4in,colback=white]
    \centering
    {\itshape vertex form}\\[0.5\baselineskip]
    \Large
    $g(x) = \bm{a}(x-\bm{h})^2 - \bm{k}$
\end{tcolorbox}

But you have to remember that $\bm{h}$ is always the 
\gap{opposite} \gap{of} \gap{what} \gap{you} \gap{see}.

\myProblemsWithContent[Find $a$, $h$, $k$ for these quadratic functions in vertex form.]
{
    $3(x-4)^2 + 5$
    \tcblower 
    $a = $\gap{3}
    \hfill 
    $h = $\gap{4}
    \hfill 
    $k = $\gap{5}
}
{
    $-3(x+6)^2 -7$
    \tcblower 
    $a = $\gap{-3}
    \hfill 
    $h = $\gap{-6}
    \hfill 
    $k = $\gap{-7}
}
