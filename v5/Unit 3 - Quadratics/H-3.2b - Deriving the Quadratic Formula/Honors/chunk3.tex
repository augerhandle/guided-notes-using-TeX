\section{The Discriminant}

The quantity \gap{$b^2 - 4ac$} is called the \gap{discriminant}.

\begin{myConceptSteps}{To calculate the discriminant of a quadratic equation\dots}
    \myStep{standard form}{Write the equation as \gap{$ax^2 + bx + c = 0$}.}
    \myStep{coefficients}{Write down the values of \gap{$a$}, \gap{$b$}, \gap{$c$}. }
    \myStep{discriminant}{Calculate the discriminant as \gap{$ D = b^2 - 4ac $}.}
\end{myConceptSteps}

\myProblems[Find the discriminant of these quadratic equations.]
{
    $9x^2 + 6x - 1 = 0$
}
{
    $ 3t^2 - 6t - 18 = -9 $
}
