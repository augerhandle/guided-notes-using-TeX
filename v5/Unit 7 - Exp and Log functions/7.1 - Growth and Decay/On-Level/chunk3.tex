\section{Euler's Number, $e$}

Facts about Euler's number, $e$:\par
\begin{itemize}
    \item $e$ is an \gap{irrational} number. \whenTEACHER{\tiny (just like $\pi$)}
    \item $e$ is approximately equal to 2.7182818285\dots  
    \item $e$ is on scientific calculators.  \whenTEACHER{\tiny (just like $\pi$)}
\end{itemize}





\myProblems[%
    Are these functions exponential growth or decay. Explain why.
]
{
    $f(x) = e^{x}$ \myAnswer{\tiny growth---$b>1$ and no $r$}
}
{
    $f(x) = e^{-3x}$ \myAnswer{\tiny decay---$b>1$ and $r=-3$ is neg.}
}
{0.5in}
