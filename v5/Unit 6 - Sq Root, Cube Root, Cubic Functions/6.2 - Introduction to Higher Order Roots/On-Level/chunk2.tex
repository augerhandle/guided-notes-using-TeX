\section{Evaluating Cube Roots on a TI-84}

You can use a calculator to evaluate $\myRoot[3]{\cdots}$. 
\myEmph{But there is no button for it!}

\begin{myConcept}{To evaluate cube roots on a TI-84\dots}
    \begin{itemize}
        \setlength\itemsep{0.4em}
        \item There is \myEmph{no button for this!} 
        \item Instead, press the \myKey{MATH} button.
        \item Use the default \myKey{MATH} menu.
        \item Select item \myKey{4}.
        \item Type in the number you want the cube root of.
        \item Select \myKey{ENTER}.
    \end{itemize}
\end{myConcept}

\myProblems[%
Evaluate the following cube roots using a TI-84 calculator.
If you get a decimal, show 3 digits after the decimal point.
]
{
    $\myRoot[3]{27}$ 
    \myAnswer{\hfill $3$ (exactly)}
}
{
    $\myRoot[3]{33}$
    \myAnswer{\hfill $\sim 3.208$}
}
{0.5in}

\myProblems
{
    $\myRoot[3]{1234.5}$
    \myAnswer{\hfill $\sim 10.727$}
}
{
    $\myRoot[3]{10999}$
    \myAnswer{\hfill $\sim 22.239$}
}
