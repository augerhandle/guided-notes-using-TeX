\section{Inverse properties}

Exponential and logarithmic functions are \gap{inverses}.
They \gap{undo} each other.

\begin{tcbraster}[
    raster equal height,
    raster columns=2,
    raster left skip = 0.25in, raster right skip = 0.25in,
    raster column skip = 0.25in
]
    \begin{tcolorbox}
        Exponentials \gap{undo} logarithms.
        \Large
        \[
            b^{log_b(A)} = A
        \]
    \end{tcolorbox}
    \begin{tcolorbox}
        Logarithms \gap{undo} exponentials.
        \Large
        \[
            log_b(b^A) = A
        \]
    \end{tcolorbox}
\end{tcbraster}

    }
    {

    }
\end{my2Problems}
