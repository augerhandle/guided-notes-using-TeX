    \begin{tcolorbox}
        \gap{Rewrite} equations using same bases when possible.
        \end{tcolorbox}
    \begin{tcolorbox}
        \centering
        $           
        \left(b^n\right)^C = \left(b^m\right)^D
        % \quad\Leftrightarrow\quad \left(b\right)^{nC} = \left(b\right)^{mD} 
        \quad\Leftrightarrow\quad nC = mD 
        $
    \end{tcolorbox}
\end{tcbraster}

When the bases of both sides are \gap{similar},
\gap{rewrite} using same base.
Then you can \gap{cancel} the base.

\begin{my2Problems}[\normalsize]{3in}[Solve these exponential equations.]
    {
        $5^{2x+7} = 5^{x+10}$
    }
    {
        $e^{5x} = e^{x+12}$
    }
\end{my2Problems}
