\section{Direct Variation}
%
\begin{minipage}{0.75\textwidth}
    \begin{itemize}
        \item \gap{Direct} variation describes a relationship between two variables.
        \item They are {\bfseries\itshape directly} \gap{proporational}.
        \item When one variable \gap{decreases}, so does the other.
        \item When one variable \gap{decreases}, so does the other.
        \item Their \gap{ratio} is {\bfseries\itshape constant}.
        \item This is a \gap{linear} function.
    \end{itemize}
    
\end{minipage}
