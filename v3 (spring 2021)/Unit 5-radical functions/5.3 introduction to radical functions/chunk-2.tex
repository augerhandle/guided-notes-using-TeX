\section{The square root parent function}
%
The square root parent function is like \gap{half} a sideways \gap{parabola}.


\begin{taggedblock}{on-level}
    \begin{myProblems2}{Convert these exponential expressions to radical expressions.}{1in}
        {
            $ x^{\frac{4}{7}}$ 
        }
        {
            $ (2ab)^\frac{1}{3}$ 
        }
    \end{myProblems2}
\end{taggedblock}
\begin{taggedblock}{pre-AP}
    \begin{myProblems2}{Convert these exponential expressions to radical expressions.}{1in}
        {
            \Large $ (7\beta\gamma)^\frac{5}{2}$ 
        }
        {
            \Large $ (11\alpha^2\beta)^\frac{1}{3}$ 
        }
    \end{myProblems2}
\end{taggedblock}


\vspace{\baselineskip}
In some cases, you might be able to \gap{simplify} the radical expressions using 
what you learned in {\scshape Lesson 5.1}.
