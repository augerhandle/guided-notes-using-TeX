\section{Function families}

A {\bfseries\itshape function family} is a set of functions \gap{look alike}. 
You already know two:
\begin{itemize}
    \item \gap{linear} functions (Their graphs are \gap{lines}.)
    \item \gap{quadratic} functions (Their graphs are \gap{parabolas}.)
\end{itemize}

In this unit, we are studing a \gap{third} function family.
\begin{itemize}
    \item \gap{square root} functions 
\end{itemize}
